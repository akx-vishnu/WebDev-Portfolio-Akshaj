# LittleCloud Baby Wear

LittleCloud Baby Wear is a specialized manufacturer of premium baby clothes for ages 1-3 years. This modern, high-performance website serves as a digital showroom and B2B inquiry portal.

## 🚀 Key Features

- **Real-time Product Search**: Instant filtering of products across all categories with auto-scroll to results.
- **WhatsApp Integration**: One-click product inquiries and contact form submissions sent directly to WhatsApp.
- **Dynamic Configuration**: Easily update content, products, and prices via `src/config.ts`.
- **Responsive Design**: Premium aesthetics optimized for all devices with smooth entrance animations.
- **Support System**: Functional FAQ section with quick navigation from the footer.

## 🛠️ Tech Stack

- **React 19** & **Vite**
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Intersection Observer API** (Visibility animations)

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## ⚙️ Configuration

The entire site content is managed via `src/config.ts`. You can change:
- Brand names and descriptions.
- Product categories and items (including images and prices).
- Currency symbols (currently set to ₹).
- WhatsApp contact number and inquiry messages.
